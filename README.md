# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

* リモートリポジトリ：ネット上に配置して複数人で共有するためのリポジトリ（個人開発も可）
* ローカルリポジトリ：開発者一人ひとりが使用するために自分のPC上に配置するためのリポジトリ

## プッシュとマージの違いは何でしょうか？

* プッシュ：ローカル上の差分をリモート上に登録する
* マージ：ローカルまたはリモート間で差分を上書きする

## コミットとプッシュの違い

* コミット：ブランチ内で作業した修正内容を登録
* プッシュ：ローカルからリモートに修正内容を登録

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

* 誰が見ても一目で分かるように書く

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

* 話し合い、以下のいずれかを決める
1. ブランチAの内容を取り込む
